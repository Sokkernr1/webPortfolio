Further information about "Dwarf Island"



In order to be able to construct a terrain mesh, the terrain generator needs a .png heightmap file, such as the one displayed here. The script then reads the grayscale value of each pixel and stores the information in a 2D-array.
This array is now the key component for the entire rest of the process.

    ⤋    ⤋    ⤋    ⤋

The array is then used in combination with the user-defined values from the Unity editor to shape the terrain in accordance with the heightmap image.

    ⤋    ⤋    ⤋    ⤋

The script then uses Perlin noise to procedurally form hills and valleys in the areas of the terrain in which the corresponding grayscale value falls in-between two user-defined values.
Afterwards, the generator applies textures according to the surface position (by checking the grayscale array) and normal vector (in order to differentiate between ground and wall).
These textures are subsequently commingled smoothly at their borders to create seamless transitions.

    ⤋    ⤋    ⤋    ⤋

Finally, the script generates trees/flowers/weeds using predefined values and options.
Trees are randomly distributed over the island. An independent script then checks their surroundings to ensure that their position makes sense and that their mesh model fits to their environment.
Flowers grow in batches/groups around the Island everywhere where there is a grass texture on the ground.
Weeds, too, are placed here and there on those parts of the island with a grass texture surface.
