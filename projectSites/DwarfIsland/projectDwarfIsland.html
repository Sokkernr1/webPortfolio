Project-Dwarf Island

× ↬ ↫

About the project:
This is my main project and not yet finalized. I intend to continue working on it to improve the conceptional details.
The game is about a dwarf (the player) who starts living on a new and unexplored island. The player decides by himself whatever he wants to do: exploring, gathering, constructing, mining or venturing into unknown ruins that lay below the island...
This project is far from being finished and will experience many huge changes in the near future. It is currently in the pre-alpha state and missing a lot of features that will be implemented bit by bit subsequently.

Features:
-| Customizable PNG heightmap defining the shape of the island itself
-| Procedural generation of hills/elevations
-| Algorithm to colour the terrain mesh accordingly with predefined textures and blending them smoothly at their borders
-| Algorithm to "randomly, but logically" place trees, spawn Flowers in defined groups and create grassland along the entire island

About the code:
Everything in this project is programmed in C# within the Unity Engine.
Since this project will change drastically in the near future, I will link the current state of the game code (24.11.2020).
If you are interested in this project and want to know how the generator works at the moment, please do not hesitate to send me an e-mail asking about the newest files of the game!
I will not go into the details of how the code works for the same reason... too many changes will be coming.
Feel free to download the files, if interested!
