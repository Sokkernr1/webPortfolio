About the bot:
I created this bot for me and my classmates to handle and distribute information like schedules. It was fun creating it since I've never before programmed with an API. I learned a lot over the course of these few days and am very happy with the results!

Supported commands:
-> -schedule {argument 1} {argument 2}
This command makes the bot send the school schedule for a defined class either as a .pdf or a .png.
{argument 1}: can be either 'me' or 'a class name'. When entering 'me' the bot will use an algorithm to figure out which class the person using the command is in using his/her roles on the server.
{argument 2}: since the schedule is returned as a .png image by default, passing 'download' in here will cause the schedule to be returned as a downloadable .pdf file

-> -rolecount {argument 1} {argument 2}
This command makes the bot count all members of a role.
{argument 1}: Any rolename that exists on the current server
{argument 2}: Entering 'list' here will return a list of all members with that role

-> -settings {argument 1} {argument 2}
This command enables server admins to configure the bot to their liking.
Configurable options include: prefix, used regEx and permissions.

-> -help
As implied by the syntax of this command, help about all commands and more information will be returned to the user.

-> -ping
This command is for checking if the bot is online and will simply return 'pong' in the chat.

About the code:
I programmed the bot with javeScript using the discord.js API. Using nodeJS I managed to easily test and deploy the bot live. It is now hosted on a pebblehost.com server and always online and ready to help!
