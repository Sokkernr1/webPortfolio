About the website:
In August 2020, I started working on this website by simply creating a .txt file on my desktop. Over time, the page became larger and I got a "vision" of what I wanted it to become.
I bought a raspberry Pi computer, set it up with a dynamic dns service and got it working, until I found out that my internet provider blocked certain traffic to prevent hostings of certain servers (webservers apparently included) so that I could only reach my website via my home network...
After quite some back and forth with my provider, I decided to drop the raspberry Pi and just rent a server to host my page.

About the code:
I created this website on my own without any major help or references. I like things simplistic and thus kept the design that way. I worked hard to make the website as responsive as possible for any resolution/screen!
Almost everything stored on this webserver is open source and can be viewed and downloaded by anybody (but I still placed a download link below anyway).
