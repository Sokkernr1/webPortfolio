My most current Projects

"Dwarf Island" the game

"Dwarf Island" is my main project. It is an open world survival RPG game in which the player starts on an island and is free to decide whatever he wants to do: farming, building or exploring.
The map is generated by a generator script which uses a heightmap to create an island and then procedurally generates hills. Afterwards flora like grass, trees and flowers are spawned and placed logically along the island.

Concept: Octagon chess

This game concept is about a chess board game which is meant to be played by 4 players simultaneously. Each type of chess figure has its own special manner to move on the octagonal board tiles of the chessfield.
